# 🏃 Strava Dashboard

Painel interativo para visualização e análise das suas atividades do Strava, com mapas, gráficos e exportação para Excel.

---

## 📋 Funcionalidades

- Autenticação OAuth2 com a API do Strava
- Busca e armazenamento local de todas as atividades
- Filtros por tipo de atividade, ano, mês, estado e município
- Mapa interativo com rotas e pontos de início
- KPIs: distância total, pace médio, FC média, kilojoules, etc.
- Gráficos de distância por mês e comparativo por ano
- Agrupamento por faixas de distância
- Zonas de frequência cardíaca (Z1–Z5)
- Spatial join com municípios brasileiros (via GeoJSON)
- Exportação para Excel (.xlsx) com abas de Atividades e KPIs
